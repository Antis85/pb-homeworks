# Домашнее задание к курсу Основы программирования

## Список заданий

### Блок 1. Типы данных и ветвление

1.1. Базовые понятия. Переменные и числа

1.2. [Логический тип и ветвление](./boolean)

1.3. Строки. Алгоритмы. Как отлаживать код

### Блок 2. Структуры данных и циклы

2.1. Массивы и базовые методы работы с ними

2.2. Циклы

2.3. Ассоциативные массивы

### Блок 3. Функции

3.1. Функция без параметров. Область видимости

3.2. Параметры и возвращаемое значение

## Требования

- браузер;
- редактор кода, например, [Atom][3];
- система контроля версий [git][4], установленная локально;
- аккаунт на [GitHub][1] или [BitBucket][2].

## Начало работы

1. Клонировать репозиторий с домашними заданиями `git clone https://github.com/netology-code/pb-homeworks.git`.
2. Перейти в папку `cd pb-homeworks`.
3. Создать репозиторий на [GitHub][1] или [BitBucket][2].
4. Добавить репозиторий в проект `git remote add homeworks %repo-url%`, где `%repo-url%` — адрес созданного репозитория.

## Решение задач

1. Перейти в папку задания, например, для первого задания `cd ./basics`.
2. Выполнить задание.
3. Опубликовать код в репозиторий homeworks с помощью команды `git push -u homeworks master`.
4. Прислать ссылку на репозиторий через личный кабинет на сайте [Нетологии][0].
